Acerca de TaskMaster

Un proyecto web pensado para organizar tu vida y practicar desarrollo frontend.

¿Qué es TaskMaster?

TaskMaster es un sistema web de gestión de tareas que permite crear, editar, organizar y visualizar tareas con prioridades, estados, etiquetas y subtareas.

Forma parte de un Proyecto Web Integrador 2026, enfocado en aplicar HTML, CSS, Bootstrap y JavaScript de forma práctica.

Características principales
  • Gestión de tareas con prioridad y estado.
  • Subtareas y progreso visual.
  • Etiquetas y filtros avanzados.
  • Estadísticas con gráfico.
  • Persistencia en localStorage.
  • Tema claro/oscuro.
Créditos

Proyecto desarrollado como parte del curso de desarrollo web.
